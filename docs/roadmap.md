# Development Roadmap

## Phase 1: Foundation

- Monorepo, Docker Compose, environment files.
- Prisma schema, seed data, SQL/ER documentation.
- API health, auth, roles, validation, Swagger.

## Phase 2: Core Operations

- Meal entry, attendance, distribution, wastage, inventory APIs.
- Flutter kitchen app screens and local sync queue.
- Dashboard KPI cards and charts.

## Phase 3: AI Nutrition

- Nutrition totals and scoring.
- Deficiency detector.
- Cost-aware weekly planner.
- Recommendation persistence.

## Phase 4: Monitoring and Reports

- School and district analytics.
- Report generation payloads.
- Export formats.
- School comparison views.

## Phase 5: IoT Expansion

- Device registry.
- Smart scale event ingestion.
- Automatic inventory adjustment.
- Ingredient consumption forecasting.
